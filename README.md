# Flashcard App

A small Node.js flashcard app that stores cards in JSON files within this directory.

## Run

```bash
npm start
```

Then open `http://localhost:3000`.

## Storage

Flashcards are stored in:

- `data/flashcards.json`

## Features

- Create flashcards
- Edit flashcards
- Delete flashcards
- Search flashcards
- Click a card to reveal its answer
